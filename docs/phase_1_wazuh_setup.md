# Phase 1: Get Wazuh Set Up

## Goal
Deploy a free local SIEM logging stack using Wazuh and Docker Compose

## Tools Used
- Wazuh
- Docker and Docker Compose
- Github
- VSCode

## Actions Taken
- Made the repository
- Created and modified .gitignore (don't want to publish certs or bloat)
- Cloned the official Wazuh Docker repository
- Generated Wazuh indexer certificates
- Started the Wazuh single-node stack and verified access on localhost
